## 🖼️ GitHub Avatar 设计方案

### 方案 1：古罗马柱 + 代码（推荐）

```
背景：深蓝色 (#1a365d)
主元素：白色古罗马柱（简化的多立克柱式）
装饰：金色 </> 符号
风格：极简、专业
```

**Gravatar 邮箱**：`catocode@catocode.dev`

### 方案 2：盾牌 + 对勾

```
背景：渐变（深蓝到金色）
```

```
背景：深蓝色

```

## 🎨 颜色方案

### 主色调
```css
--cato-blue: #1a365d;      /* 深蓝 - 信任、严肃 */
--cato-gold: #d4af37;      /* 金色 - 品质、古罗马 */
--cato-green: #10b981;     /* 绿色 - 验证通过 */
--cato-red: #ef4444;       /* 红色 - 错误、警告 */
--cato-gray: #6b7280;      /* 灰色 - 次要信息 */
```

### 使用场景
- **Logo**: 深蓝 + 金色
- **成功状态**: 绿色
- **失败状态**: 红色
- **文档**: 深蓝 + 灰色

## 📝 Git 配置

### 容器内 Git Identity

```bash
# 在 Dockerfile 或 manager.py 中设置
git config --global user.name "CatoCode"
git config --global user.email "catocode@catocode.dev"
git config --global user.signingkey "catocode@catocode.dev"
```

### Commit 格式

```
feat: add null check for email field

Fixes #123

Co-Authored-By: User Name <user@example.com>

---
Generated by CatoCode v1.0
Evidence: /tmp/evidence-before.txt, /tmp/evidence-after.txt
```

### Commit 签名

所有 commit 都应该包含：
- Author: CatoCode
- Committer: CatoCode
- Co-Author: 用户（从环境变量获取）

## 🖼️ 生成头像的方法

### 方法 1：使用 Gravatar

1. 注册 `catocode@catocode.dev` 邮箱
2. 上传头像到 Gravatar
3. GitHub 会自动使用 Gravatar 头像

### 方法 2：使用 GitHub 默认头像

GitHub 会根据邮箱生成默认头像（identicon）

### 方法 3：自定义头像（推荐）

创建一个 512x512 的 PNG 图片：

**使用 AI 生成**：
```

"A minimalist logo for a code guardian AI. Ancient Roman column in white
on dark blue background, with golden code brackets </> symbol.
Professional, clean, modern meets classical. 512x512 pixels."
```

**或使用在线工具**：
- Canva

- Adobe Express
